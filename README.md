# KhaKhau - Cuisine Management System

## Project Description
A web-based cuisine management application with admin panel for managing menu, orders, and users. Developed as a personal project to practice backend development with C# and .NET technologies.

## Technologies Used
- **Backend**: C# , .NET Framework / ASP.NET
- **Frontend**: HTML, CSS, JavaScript, jQuery
- **Database**: Microsoft SQL Server (not included in repo)
- **IDE**: Visual Studio

## Features
- Admin panel for managing cuisine data
- Data entry and processing functions
- Basic CRUD operations
- Responsive web interface

## Project Structure
- `KhaKhau.sln` - Main solution file
- `KhaKhau/` - Main project folder

## How to Run
1. Open `KhaKhau.sln` with Visual Studio
2. Restore NuGet packages
3. Update connection string to your SQL Server
4. Run the project
